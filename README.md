# Data Analyze CMS

imooc 流量统计分析课程代码

## Setup Step

1. Install [MAMP-PRO](https://www.mamp.info/en/mamp-pro/)

2. Download (GXCMS source code)[http://www.gxcms.org/]

3. 在CMS后台管理系统中添加采集基本`calc.js(jquery)`

## Start Server

1. Install dependencies

```bash 
glide up        # glide was required
```

2. start

```
go run main.go
```
